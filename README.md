# ⚖️ Progetto Blockchain Notarile
Sistema sperimentale per la notarizzazione di contratti e testamenti digitali.

## 🛠️ Caratteristiche Tecniche
- **Immutabilità:** Basata su algoritmo SHA-256 per garantire l'integrità dei documenti.
- **Data Certa:** Timestamping automatico per ogni transazione.
- **Sicurezza Cyber:** Architettura "Hash-only" per rispettare la privacy (GDPR).

## 📜 Conformità Legale (Italia)
Il progetto mira a soddisfare i requisiti del:
1. **Art. 8-ter DL 135/2018:** Validazione temporale elettronica.
2. **CAD (Codice Amministrazione Digitale):** Integrazione futura con Firma Elettronica Avanzata.
3. **Codice Civile:** Supporto alla conservazione di testamenti e atti privati.
